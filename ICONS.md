# Создание иконок для PWA

Для полноценной работы PWA приложения необходимо создать две иконки:

- `icon-192.png` - 192x192 пикселей
- `icon-512.png` - 512x512 пикселей

## Способы создания иконок

### Способ 1: Онлайн-генератор
1. Перейдите на https://realfavicongenerator.net/ или https://www.pwabuilder.com/imageGenerator
2. Загрузите изображение логотипа (минимум 512x512)
3. Скачайте сгенерированные иконки
4. Переименуйте их в `icon-192.png` и `icon-512.png`
5. Поместите в корневую директорию проекта

### Способ 2: Вручную
1. Создайте квадратное изображение с логотипом приложения
2. Используйте графический редактор (Photoshop, GIMP, Figma) для создания:
   - Иконки 192x192 пикселей
   - Иконки 512x512 пикселей
3. Сохраните как PNG с прозрачным фоном (если нужно)
4. Поместите файлы в корневую директорию проекта

### Способ 3: Использование простого SVG (временное решение)
Можно временно использовать простой SVG файл, конвертировав его в PNG через онлайн-конвертер.

## Рекомендации по дизайну иконок

- Используйте простой, узнаваемый символ
- Избегайте мелких деталей
- Используйте контрастные цвета
- Убедитесь, что иконка хорошо выглядит на тёмном и светлом фоне
- Следуйте рекомендациям Apple для иконок приложений

## Проверка

После добавления иконок:
1. Откройте приложение в браузере
2. Проверьте вкладку Application → Manifest в DevTools
3. Убедитесь, что иконки загружаются без ошибок
